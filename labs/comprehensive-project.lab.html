Capstone Project

Comprehensive Project Lab

All-in-One: Security, VLANs, Trunking, OSPF, NAT, and ACL

Objective

Combine all learned CCNA technologies into one cohesive environment. You will configure basic security (SSH, passwords, banners), Set up VLANs and Trunking, Route between VLANs (ROAS), Establish Dynamic Routing (OSPF), configure NAT, and secure the network using Access Control Lists (ACLs).

Lab Topology

Device Interface IP Address Subnet Mask Description
R1 Gi0/1.10 192.168.10.1 255.255.255.0 Gateway for VLAN 10 (Sales)
R1 Gi0/1.20 192.168.20.1 255.255.255.0 Gateway for VLAN 20 (IT)
R1 Gi0/0 203.0.113.1 255.255.255.252 WAN / Internet Link (OSPF & NAT Outside)
SW1 Gi0/1 - - 802.1Q Trunk Link to Router
SW1 Fa0/1 - - Access Port - VLAN 10 (Sales PC)
SW1 Fa0/2 - - Access Port - VLAN 20 (IT PC)
Remote Host Extern 8.8.8.8 255.0.0.0 Internet Destination

Task 1 - Basic System & Security Setup (Router & Switch)

Apply hostname, domain name, passwords (Console, VTY, Enable), banners, and enable SSH access.

Router> enable
Router# configure terminal
Router(config)# hostname R1
R1(config)# no ip domain-lookup
R1(config)# ip domain-name ccnaproject.lab
R1(config)# crypto key generate rsa modulus 2048
R1(config)# username admin privilege 15 secret cisco123
R1(config)# enable secret class
R1(config)# banner motd # WARNING! UNAUTHORIZED ACCESS PROHIBITED #

R1(config)# line console 0
R1(config-line)# password cisco
R1(config-line)# login
R1(config-line)# exit

R1(config)# line vty 0 4
R1(config-line)# login local
R1(config-line)# transport input ssh
R1(config-line)# exit

Task 2 - Core Switching (VLANs & Trunking) on Switch

Create VLANs and a trunk port facing the router.

Switch(config)# hostname SW1
SW1(config)# vlan 10
SW1(config-vlan)# name Sales
SW1(config-vlan)# vlan 20
SW1(config-vlan)# name IT
SW1(config-vlan)# exit

SW1(config)# interface gigabitEthernet 0/1 ! Connection to Router
SW1(config-if)# switchport mode trunk
SW1(config-if)# exit

SW1(config)# interface fastEthernet 0/1
SW1(config-if)# switchport mode access
SW1(config-if)# switchport access vlan 10
SW1(config-if)# exit
SW1(config)# interface fastEthernet 0/2
SW1(config-if)# switchport mode access
SW1(config-if)# switchport access vlan 20

Task 3 - Inter-VLAN Routing (Router-on-a-Stick)

Configure sub-interfaces on Router1 to act as gateways for the VLANs.

R1(config)# interface gigabitEthernet 0/1
R1(config-if)# no shutdown
R1(config-if)# exit

R1(config)# interface gigabitEthernet 0/1.10
R1(config-subif)# encapsulation dot1q 10
R1(config-subif)# ip address 192.168.10.1 255.255.255.0
R1(config-subif)# exit

R1(config)# interface gigabitEthernet 0/1.20
R1(config-subif)# encapsulation dot1q 20
R1(config-subif)# ip address 192.168.20.1 255.255.255.0

Task 4 - Dynamic Routing (OSPF)

Enable OSPF so the internal subnets are advertised to the WAN router (assume a WAN router exists on Gi0/0). 

R1(config)# interface gigabitEthernet 0/0 ! WAN Interface
R1(config-if)# ip address 203.0.113.1 255.255.255.252
R1(config-if)# no shutdown
R1(config-if)# exit

R1(config)# router ospf 1
R1(config-router)# router-id 1.1.1.1
R1(config-router)# network 192.168.10.0 0.0.0.255 area 0
R1(config-router)# network 192.168.20.0 0.0.0.255 area 0
R1(config-router)# network 203.0.113.0 0.0.0.3 area 0

Task 5 - NAT (PAT Overload) Setup

Translate inside IP addresses (VLANs 10 & 20) to the Router's outside interface (Gi0/0) public IP.

R1(config)# interface gigabitEthernet 0/1.10
R1(config-subif)# ip nat inside
R1(config-subif)# exit
R1(config)# interface gigabitEthernet 0/1.20
R1(config-subif)# ip nat inside
R1(config-subif)# exit
R1(config)# interface gigabitEthernet 0/0
R1(config-if)# ip nat outside
R1(config-if)# exit

R1(config)# access-list 1 permit 192.168.10.0 0.0.0.255
R1(config)# access-list 1 permit 192.168.20.0 0.0.0.255
R1(config)# ip nat inside source list 1 interface gigabitEthernet 0/0 overload

Task 6 - Network Security (Extended ACL)

Deny VLAN 10 (Sales) from pinging a specific external destination (e.g., 8.8.8.8) but allow HTTP and all other traffic.

R1(config)# ip access-list extended SECURE_SALES
R1(config-ext-nacl)# deny icmp 192.168.10.0 0.0.0.255 host 8.8.8.8 echo
R1(config-ext-nacl)# permit ip any any
R1(config-ext-nacl)# exit

R1(config)# interface gigabitEthernet 0/1.10
R1(config-subif)# ip access-group SECURE_SALES in

Task 7 - Verification Commands

R1# show run
R1# show ip interface brief
R1# show ip ospf neighbor
R1# show ip nat translations
R1# show access-lists
SW1# show vlan brief
SW1# show interfaces trunk
